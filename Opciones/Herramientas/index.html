Nuestras herramientas

Gracias a los conocimientos obtenidos tras finalizar el grado de Inteligencia de Negocios, se ha podido llevar a cabo la elaboración de este proyecto, aplicando la visión y las técnicas del análisis de datos. Para su elaboración se ha hecho uso de varias herramientas que se describirán a continuación.

Empezamos con el lenguaje de programación Python. Este es realmente conocido y utilizado para el análisis de datos, la automatización de operaciones, el machine learning y la minería de datos. Este lenguaje tiene numerosas librerías que permiten el acceso de funcionalidades específicas con las que poder resolver diversas funciones. En este proyecto se han utilizado las diferentes librerías dentro de los desarrolladores de Python como Pandas, que es una librería muy popular de código abierto para el ámbito de Machine Learning y ciencia de datos. Así mismo, se utiliza la librería de Numpy especializada en el análisis de datos y en el cálculo numérico. Otra de las librerías de Python que se han utilizado en el proyecto es Matplotlib, especializada en la creación de gráficos en dos dimensiones. A su vez, se hace uso de scikit-learn que es una librería de Python para hacer análisis predictivos, que incluyen clasificadores, algoritmos de clusterización, etc, y está basada en Numpy, Matplotlib y Scipy, de modo que es fácil reaprovechar el código que use estas librerías. Se ha hecho uso a su vez de la biblioteca Keras, que es capaz de crear redes neuronales.

Seguimos con MongoDB, es un sistema de gestión de base de datos NoSQL orientado a documentos donde se almacenan los datos y de código abierto. Es una base de datos no relacional (NoSQL) debido a que almacena y accede a los datos de una forma estructurada. Este se caracteriza por su flexibilidad y escalabilidad al trabajar, debido que es sencillo e intuitivo, a su vez permite usar diferentes lenguajes de programación, como Python, con el cual se va a trabajar en este proyecto. Se utiliza para la elaboración de este proyecto, debido a que MongoDB permite realizar consultas pudiendo hacer búsquedas por campos, tiene una gran cantidad de almacenamiento que le permite actuar como sistema de archivos y facilita la posibilidad de crear, administrar y rehacer índices, ofrece alto rendimiento, alta disponibilidad y escalado automático.

Debido a que la finalidad de este proyecto se basa en el desarrollo de una web con datos en tiempo real para poder predecir con precisión la mejor ubicación en la que llevar a cabo el desarrollo de una tienda, se va a hacer uso de Pyscript. Pyscript es un entorno de desarrollo integrado de Python, que ha supuesto el cambio permitiendo que se pueda escribir y ejecutar código de Python en un navegador, como en el caso de la interfaz de HTML, que es un lenguaje que sirve para el desarrollo de páginas web. Por lo tanto, esta herramienta tan novedosa, va a ser fundamental para la elaboración del proyecto, debido a que permite la introducción en la página web de funcionalidades de Python para el estudio y el análisis de los datos a tiempo real.

Para dar formato y diseño a la página web se ha hecho uso de CodePen, que es una comunidad para desarrolladores web, donde se puede compartir, actualizar, ver, comentar, diseñar y crear códigos web. Su principal enfoque es un lenguaje del mundo front-end como HTML, CSS y JavaScript, por lo que es perfecto para el caso de este proyecto, ya que es una plataforma que ofrece el código de diferentes plantillas y funcionalidades realmente útiles ofreciendo la posibilidad de exportar dicho código y usarlo para llevar a cabo el diseño de la página web.

Como acabamos de comentar, para la elaboración de la página web se hace uso del lenguaje HTML, acompañado de los lenguajes JavaScript y CSS. HTML sirve para describir cómo debe mostrarse el contenido en la web asignando etiquetas al contenido. JavaScript es el lenguaje de programación de la web, este permite a los desarrolladores crear páginas web interactivas y dinámicas. Las hojas de estilo en cascada (CSS) son una forma de describir el aspecto, el diseño y el comportamiento de una página web. Controla cómo se presentan las páginas HTML en un navegador, se basa en añadir atributos de clase a las etiquetas HTML para cambiar su aspecto o formato.

Para la ejecución y edición del código de este proyecto se ha hecho uso de Visual Studio Code es un editor de código, que permite numerosos lenguajes de programación, como Python, HTML, JavaScript y CSS, usados para el desarrollo de este proyecto.

Para subir y almacenar repositorios se ha hecho uso de GitHub. Este es un servicio basado en la nube que permite subir repositorios de código y que aloja un sistema de control de versiones.

Y por último, el sitio de referencia que se ha utilizado para obtener los datos de entrenamiento ha sido Kaggle. Este es esencial a la hora de tratar con desafíos relacionados con el Machine Learning, ya que es un sitio repleto de casos de estudio, desafíos y retos donde todos pueden intentar resolverlos con sus distintos modelos de predicción para poder desarrollar nuestro algoritmo.
